Project Idea

Video Game Organiser - Product Name: Gamers Heaven

Overview

As a PC gamer, I find it difficult to find the right game to play amongst so many different shortcut icons unsorted, messy and even sometimes too small to read text on steam. Many gamers themselves are frustrated by having to switch between numerous game providers in order to locate the appropriate game to play.

The idea of a video game organiser came to mind to cater for casual and experienced video game (PC) players. I would aggregate all video gaming engines and platforms exclusive to PC content as consoles already have this feature within their built-in games library only which is the downside for PC gamers having this tool at their fingertips. All of these configurations could be found in my organiser. When you link your accounts to this software, the games you own will be sorted under your own umbrella and there will be many different navigation options for when you want to buy a new game or organise your games by most hours played or alphabetical order for those who want quick access to their content.

Motivation

This gaming organisation tool will surely increase productivity as well as the overall quality of life aspects. A non-Steam game, such as Overwatch or Call of Duty Vanguard, may be imported into Steam software to launch however, this merely opens the launcher, making it cumbersome for the gamers to click the play button again if they would like to get right into the action. This would not only allow for more quick launching capabilities, but it will also allow for more personalisation for individuals who want to show off their games to their friends. Similarly to their collection of disc-based games for consoles as means to boast all over their friends!

Description

The features that will be included in this game organiser would be what any unorganised person would ever want. It would import all existing games a user owns and would display them on a menu selection screen. Similar to the GeForce Experience software, it would scan folders within your Program Files as a default setting. With multiple other drives and folders, you may have can provide the directory for the software to find those games you have installed. Also, as you launch the game it will also open up the launcher in the background that the game has to use. Overall, an efficient one-click play button would load up all your saved settings you’ve already entered. To reduce the number of clicks and misclick attempts, the program will also inform you of what is currently being executed such as multiple game updates or if you’re downloading a new game.

The software would automatically check the server for any potential updates to ensure that you have the latest version installed and the service will provide you with the latest news (patch notes, events, etc). With one click, on a chosen game, it would provide you with all the relevant information before you press start to play. Not only would it search the games you have already installed but it will also (try) find games with the most hours played, as this would form another category sorter. The program will automatically sort the last played game, most populated game with the current players in your server region and in alphabetical order. Additionally in these games, it would also track your configured settings and will give you the option to change them within the menu.

The most difficult part of this project would be to allow the major game providers to advertise their content on my software. They would be allowed to have a separate section for their promotions of current or new games on the navigation bar named “Games Store”. For the time being, the user would have to use this program to open up the launcher for the preferred game or from the web on the store page (open launcher). Cooperating with many different game companies would provide the greatest challenge, on the ease of use for the players to browse all stores within one software page.

Currently, players would have to browse many game stores with their own specific services to find new games to try and or buy. However, if this were to be implemented, given enough time and communication with other companies to allow authorisation of their products and services, it could make consumers more likely to explore all video game genres and game creators all in one place.

Our current system for custom-built PCs and even pre-built PCs, including the generic gaming laptop, can have optimized gameplay with a tuning feature. For those with Geforce Graphic Card and AMD graphic cards, you would have the same options for optimizing gameplay so you don’t need to worry about manually configuring it yourself all you have to do is click “optimize”. There would be an optimization feature that would allow you to adjust via a slider for whether you prefer quality over performance, for example. This would implement user preferences and could be disabled with a checkbox in the settings menu if you do not like optimizing your gameplay or if you prefer using your own way of optimizing gameplay. It would be completely customizable to the user and will be always pushing for more personalised options.

Tools and Technologies

There would be a recommended configuration requirement for both the hardware and software. For hardware, they would need a Geforce Graphics Card installed or an AMD Graphics Card for the importance of game performance options. This does not mean you can’t have a different GPU but only a recommendation if you wish to have the best optimisation. As well as, the minimum of at least 8-16 GB of RAM. The CPU should be Intel I5 6th Gen and newer and/or equivalent. There is no absolute maximum storage as this is limitless to the user's preference but a minimum for gamers would at least be 250GBs.

The software side is intended for a library of games, but will not have such a large file size as the software is meant to be only for grabbing games already installed so there is no need to worry about any storage issues. The GPU drivers would have to be installed including being logged into your accounts. This allows for smooth transitions into gameplay rather than having to sign in every time you launch an application like Epic Games.

Outcome

The original problem of sorting out different games in different launchers does make it difficult for users as well as manage different shortcuts on a desktop home screen as it increases the visual clutter. Icons that are not needed would go and it would give gamers less of a headache from sorting games that aren’t even installed be on their games library. The outcome of the game organiser would be to provide a more efficient, more capable, less user input launcher that would be more compatible with most games to this day.

The importance of this development of this software would make life less of a hassle for users for a clear overview section that would display current games downloaded and those yet to be downloaded. Having to organize their own system in regard to what content is seen and that which is hidden. This could even be used for gamers who wish to showcase their massive game collection to other players or friends or on social media.

The issue that comes to mind is; would gamers even want this software? Do they want a one-page overview of their game software investment? Well, of course! It’s a customizability feature that would allow every gamer to choose how this program/software looks and create their own theme at the user's discretion. Furthermore, it provides services that aren’t even implemented yet on other platforms which can instantly load up any game of their choosing without the need to choose to use the launcher to confirm the game they are playing. This will be hidden away in the background so the user doesn’t have to provide more input again for a game they already chose to be confirmed again with the launcher.

Another issue that was brought into the picture is why use hardware optimization within the software when there is already GPU optimization software and drivers? Being able to already optimize your game is already implemented when the game is not running. However, the reason for integrating these software options and capabilities is to allow a more diverse range of options for the user. Optimizing gameplay is the most important factor for gamers as they don’t want to deal with recommended settings that are sometimes offered by the PC system itself that will give them terrible frames per second and even slow down the computer.

With more resources being inputted into the game itself, gamers would also want enough power to run simple programs like their web browser for a second monitor. They don’t have to include their GPU hardware and software into this program when they first download the game organiser. If they wish to, there will be an option for that which again comes back to the idea of the user having preferences.
